Universal Remote

Technologies Arduino, C++
Source Code https://github.com/brendan-w/arduino_ir

This IR remote control was made to operate any function of any device. Using an Arduino Nano, tables of IR codes can be programmed into the flash memory. When a button is pressed, the remote will iterate through IR codes from various manufacturers. While the code tables are by no means complete, several of the market-leading TV manufacturers are already compiled into the software.

Internally, IR codes are stored in a format similar to Pronto Hex. However, to conserve memory, they are stored using a dictionary and byte-length based compression, and are decompressed upon retrieval. This technique was inspired by the TV-B-Gone, a similar project that sends OFF commands. As time goes on, I intend to expand the command library to make it control a wider array of devices.

Hardware

The outer casing is made from lightly polished 3/4" copper pipe. This not only provides a solid frame for the internal components, but also acts as a grounding bus. There are two function buttons, one bank-select button, and a ten segment LED strip.

The topmost (brown) button is for bank selection. This cycles through various sets of IR functions (power, volume, channel, etc...), which are displayed on the LED strip. Each bank can have two functions that are activated by the corresponding white button (such as power ON and power OFF).

The top portion of the remote can be removed from the handle, revealing the electronics. Batteries are mounted within the handle, as well as the power switch at the rear of the remote.

A closeup of the controls while powered. The LED strip shows that the remote is set for bank one (currently mapped to power control), which means that the two white buttons will send power OFF and power ON commands, respectively.
