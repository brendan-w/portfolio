One of my many hobbies is making laser light shows. To facilitate this, I have crafted my own software for drawing and transmitting lasers graphics. This project is currently a collection of backend libraries and binaries which can form the bones of laser show GUIs, or creative applications. It features basic point manipulation functions, an ILDA file parser (laser graphics file format), and a ZeroMQ daemon for operating the hardware.

In order to transmit graphics to the laser projector, a little bit of hardware is needed. This hardware typically comes in the form of a digital to analog adapter (DAC). Currently, this project is setup to communicate with the EtherDream Laser DAC developed by Jacob Potter.

Older Work

In order to control the projector, I wrote custom software in visual basic (I was a young programmer). This provided an interface in which to draw vector laser. Tools such as point manipulation, interpolation, and effects are all built in to the software. Once drawn, the image can then be sent to the projector via that parallel port.

This program also features an effects stack. This lets the user sequentially apply modifiers to the drawn image. Effects such as translation, scale, and duplication (both linear and radial) can be applied to produce the desired effect. Some of these effects even provide rudimentary animation capabilities.
