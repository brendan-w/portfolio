It all started with this TED talk by Chris Domas. During the presentation, he shows some elegant visualizations of raw binary data. The first is digraph, with the value of any given byte (n) on the X-axis, and the value of the next byte (n+1) on the Y-axis. This creates a beautiful frequency map of the byte patterns in a given file, and it is remarkably good at distinguishing the fileâ€™s contents. Of course, this was simply too cool, and I had to make my own (visit the "Run" link). To use it, drag and drop your file into the page, and it will assemble your very own visualization.

Chris Domas has released a demo version of his software ..cantor.dust... He also gave a very good Derbycon talk, where he goes into more detail about each visualization.

While my software is only a simple web app, I fully intend to continue exploring the topic of data visualization for reverse engineering. I am currently playing with openFrameworks as a possible display framework for future iterations of this project.
