C-Graph

Technologies NodeJS, D3, Git
Run http://cgraph.herokuapp.com/
Source Code http://github.com/brendanwhitfield/cgraph
Collaborators Will Paul
License GPL v2

C-Graph is a GitHub-integrated call graphing utility for C. After providing a url for a GitHub repository, C-Graph will clone the repo, search it for C files, and construct a static call graph. The graph is interactive, and allows users to traverse the code as it would be executed, rather than as it is written.

The graph itself is a force-directed graph written by Will Paul using D3.js. It represents the code at both the file and function level. File nodes can be expanded into sets of smaller function nodes, with arrows depicting the calls from one function to the next. Selecting a function or file will also display its raw C code.

Curiously, C-Graph uses a custom C parser to construct its graphs. Many pre-existing graphing utilities were rejected because they failed to deliver some piece of data or functionality. In particular, the ability to parse a project without its dependant libraries was a common failing point. So, instead of doing a full syntax parse (that requires numerous headers), this system only searches for function calls and definitions. Once located, they can be linked by name, location, and storage class to generate a call graph. The C language was chosen specifically for its ease and simplicity in linking.
