Execute JS

Technologies Javascript, UglifyJS2
Run http://people.rit.edu/bcw7044/tools/exejs/
Source Code https://github.com/brendanwhitfield/execute-js
License LGPL v3

This tool visualizes Javascript as it executes in a web browser. After supplying the script, the user can watch the computer execute each line of code. As it runs, a heatmap is built which displays the total number of times a each line was executed. One-off commands can also be issued, allowing for experimentation with larger libraries.

Examples

Process

The execution detection is done by injecting callbacks before (or in) every javascript statement. This is done with UglifyJS2, which creates a syntax tree from the users Javascript. This tree is then modified by adding new nodes which define the various callbacks, and is recompiled to Javascript text. Here is an example input script, along with its post-processed counterpart. 


for(var i = 0; i < 70; i++)
{
	console.log("i is: " + i);

	if(i % 2 === 0)
		console.log("multiple of 2");
	else if(i % 5 === 0)
		console.log("multiple of 5");
	else if(i % 7 === 0)
		console.log("multiple of 7");
	else
		console.log("else");
}
			

for (var i = 0; __exeJS__.l(1, i < 70); i++)
{
	__exeJS__.l(3);
	console.log("i is: " + i);
	
	if (__exeJS__.l(5, i % 2 === 0))
	{
		__exeJS__.l(6);
		console.log("multiple of 2");
	}
	else if (__exeJS__.l(7, i % 5 === 0))
	{
		__exeJS__.l(8);
		console.log("multiple of 5");
	}
	else if (__exeJS__.l(9, i % 7 === 0))
	{
		__exeJS__.l(10);
		console.log("multiple of 7");
	}
	else
	{
		__exeJS__.l(12);
		console.log("else");
	}
}
